# Cloud Computing Labs Repository

## Overview
This repository contains my **Cloud Computing Assignments** with hands-on Azure lab tasks designed to build practical cloud computing and cloud administration skills.

## Objective
The objective of these labs is to gain practical experience in core Azure services by creating and managing cloud resources, implementing security and governance policies, managing identities and access control, configuring networking, deploying compute resources, and monitoring Azure services effectively.

## What I Learned
- Deploying and managing virtual machines and networks  
- Securing resources using RBAC, Key Vault, and Azure policies  
- Managing identities, roles, and subscriptions  
- Deploying web apps and container-based applications  
- Monitoring and maintaining Azure resources
  
## Labs Included
### Fundamental Labs
- Create a Virtual Machine  
- Create a Virtual Network  
- Create a Storage Account  
- Create a Web App  
- Implement Azure Key Vault  
- Secure Network Traffic

### Advanced Labs
- Manage Entra ID Identities

- Implement Virtual Networking  
- Manage Virtual Machines  
- Implement Azure Container Instances  
- Implement Azure Container Apps  
- Implement Web Apps  
- Implement Monitoring
